# Data Analysis System

A simple web-based CSV data analysis system built using Node.js, Express, and JavaScript.

Users can:

- Upload CSV files
- Select columns for analysis
- Calculate:
  - Average
  - Minimum
  - Maximum
- View complete CSV data in tabular format

---

# Technologies Used

- Node.js
- Express.js
- Multer
- CSV Parser
- Express Session
- HTML
